# IoT Motor Monitoring Dashboard

A simple IoT-based dashboard built using **Node.js**, **Express**, **SQLite**, **HTML**, **CSS**, and **JavaScript** to monitor industrial motors across zones.  
The system displays **motor status**, **sensor readings**, and **visual graphs** for temperature, vibration, and sound.
